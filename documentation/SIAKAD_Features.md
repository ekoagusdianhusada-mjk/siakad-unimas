# SIAKAD System Documentation

## Overview
The SIAKAD (Sistem Informasi Akademik) system provides comprehensive features for managing academic data, including faculty, program study, campus information, and student data access. This document outlines the capabilities and functionalities offered by the system.

## Features

### 1. Adding Faculty
- **Functionality**: Admin users can add new faculty members to the system.
- **Data Fields**:
  - Name
  - Faculty ID
  - Department
  - Email
  - Phone Number

### 2. Program Study Management
- **Functionality**: The system allows for the management of various programs of study.
- **Data Fields**:
  - Program Name
  - Program Code
  - Duration
  - Degree Type
- **Operations**: Add, edit, and delete programs of study.

### 3. Campus Data Management
- **Functionality**: Manage multiple campus locations and their respective details.
- **Data Fields**:
  - Campus Name
  - Campus Code
  - Address
  - Phone Number
- **Operations**: Add, edit, and delete campus information.

### 4. Student Data Access
- **Functionality**: Students can access their academic records and profiles.
- **Data Available**:
  - Student ID
  - Name
  - Courses Enrolled
  - Grades
  - Attendance Records

### 5. User Roles and Permissions
- The system supports various user roles, including:
  - Admin
  - Faculty
  - Student
- Access permissions are defined based on user roles to ensure data security.

### Conclusion
The SIAKAD system is designed to streamline academic data management, enhancing the efficiency and productivity of educational institutions. Proper understanding and utilization of the features outlined in this documentation will aid users in effectively navigating the system.
